Make Your Own Tile Pack

  • Create a folder called "2048 Custom Tile Pack"
  • Find 11 Images
  • Convert them to JPG using any tool
  • Name them the following Names:

    2.jpg, 4.jpg, 8.jpg, 16.jpg, 32.jpg, 64.jpg, 128.jpg, 256.jpg, 512.jpg, 1024.jpg, 2048.jpg

  • Upload the folder, to any file hosting service.
    (as long as it will link to a folder directly)
  • copy the folder's link
  • Make sure that the link ends with /
  • Paste the link on custom2048.github.io
  • Click Submit, and have fun!
